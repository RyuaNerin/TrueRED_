# TrueRED_

## 사용법
빌드 전에 Nuget Package Manager를 열고 아래 명령어를 입력해주세요. 
```
nuget install packages.config
```

현재 개발 및 프로젝트 시작을 위한 [샘플 모듈과 스트링셋](https://github.com/Twitterian/TrueRED_/releases/tag/0.1)을 제공하고 있습니다.

해당 프리 릴리즈는 예고없이 삭제될 수 있습니다.
